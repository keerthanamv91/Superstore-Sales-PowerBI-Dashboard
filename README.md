📊 Superstore Sales Dashboard – Power BI Project

📌 Project Overview

The Superstore Sales Dashboard is an interactive Power BI project designed to analyze retail sales data and generate actionable business insights.
This dashboard transforms raw transactional data into meaningful visual reports to evaluate sales performance, profitability, regional contribution, and customer segment behavior.

The project demonstrates skills in:

✨ Data Cleaning (Power Query)

✨ Data Modeling

✨ DAX Calculations

✨ Data Visualization

✨ Business Insight Generation

🎯 Project Objectives

✨ Analyze overall sales and profit performance

✨ Identify high-performing regions

✨ Evaluate segment-wise profit contribution

✨ Track sales trends over time

✨ Compare category and sub-category performance

✨ Support data-driven decision-making

📂 Dataset Description

The dataset contains retail transactional data with the following key columns:

✨ Order ID

✨ Order Date

✨ Ship Date

✨ Ship Mode

✨ Customer Name

✨ Segment (Consumer, Corporate, Home Office)

✨ Region (Central, East, South, West)

✨ Category

✨ Sub-Category

✨ Product Name

✨ Sales

✨ Quantity

✨ Profit

✨ Payment Mode
